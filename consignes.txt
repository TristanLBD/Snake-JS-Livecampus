Dans un navigateur réalisez un jeu de Snake dans un Canvas
Règle 1 : le corps suit la tête ✅
Règle 2 : si on mange un fruit le corps grandi de 1 on marque dix points ✅ 
Règle 3 : si la tête mange le corps c’est perdu ✅
Règle 4 : le plateau est un donut (le haut communique avec le bas, la gauche avec la droite) ✅
Lorsque la partie est terminée :
    Vous affichez le tableau des 20 plus grands scores ✅
    Si le joueur entre dans le top 20 vous lui proposez un formulaire contrôlé demandant son pseudo (minimum 6 caractères, maximum 20) ✅
    Si le tableau est vide vous l’initialiserez à partir des données fournies par une API ✅
    Le tableau des meilleurs scores sera conservé dans le stockage local ✅
    L’architecture de votre code devra être modulaire, vous devrez utiliser les concepts de la POO. ✅
    Votre code doit être clair et compréhensible ✅
    Aucun Framework JS autorisé. Bootstrap, Tailwind ou Sass autorisés ✅
